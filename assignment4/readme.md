Endpoints:
GET /movies: List movies

POST /movies: Create a movie
Use the following x-www-form-urlencoded fields:
    title: <name>
    year: XXXX
    actors: Actor1, Actor2, Actor3, Actor4...

GET /movies/<uuid>: View specific movie

DELETE /movies/<uuid>: Delete a movie
